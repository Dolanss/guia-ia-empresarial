# Guia de Uso de IA na Empresa

Um guia prático para usar ferramentas de IA (Lovable, Claude, ChatGPT e similares) de forma eficaz no dia a dia. Feito para quem quer parar de ficar preso no loop de "gera, quebra, gera de novo" e começar a entregar resultado de verdade.

## Por que isso existe

IA generativa não é mágica — é uma ferramenta. Como qualquer ferramenta, ela produz lixo se você não souber usar. Este guia existe para encurtar a curva de aprendizado e evitar os erros mais comuns.

## Início rápido

Se você tem 10 minutos:
1. Leia [`guias/01-fundamentos.md`](guias/01-fundamentos.md) — entenda o básico
2. Leia [`guias/02-prompts.md`](guias/02-prompts.md) — aprenda a estrutura de um bom prompt
3. Copie o template [`templates/contexto-projeto.md`](templates/contexto-projeto.md) e preencha para seu projeto atual

Se você usa Lovable especificamente, vá direto para [`guias/03-lovable.md`](guias/03-lovable.md).

## A regra de ouro

> **IA não substitui julgamento. Ela amplifica o que você já sabe.**
> Se você não consegue avaliar se o resultado está correto, o problema não é a IA — é que você não tem conhecimento suficiente para usar a ferramenta naquele contexto ainda.
